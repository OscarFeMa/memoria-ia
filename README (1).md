# 🪸 Memoria Coral — Grafo de Pensamiento IA

Sistema de visualización de memoria distribuida entre agentes de inteligencia artificial.  
Muestra en tiempo real los registros vigentes almacenados en Supabase, filtrados y organizados en un grafo interactivo.
